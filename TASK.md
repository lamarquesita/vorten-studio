# Build Vorten Studio Website

Replicate the design and layout of kargul.studio but rebrand as **Vorten Studio**.

## Site Structure (single index.html with inline CSS + JS)

### Header
- Floating pill nav at top center with logo + "Recent work" link
- Logo: simple "V" mark in a circle (SVG)

### Hero Section
- Badge: "Helped raise $440M+ in 2025. 70+ completed projects."
- H1: "A full-stack design and dev partner for high-growth startups"
- Subtitle: "A quality-obsessed design and development partner that moves fast and actually gets things shipped."
- Two CTA buttons: "Book Intro Call" (dark) and "Send Message" (white with Telegram icon)

### Showcase Gallery
- Horizontal scrolling gallery of project screenshots (use placeholder gradient cards)
- Auto-scrolling marquee effect

### Trusted By
- "Trusted by startups and industry leaders." heading
- Row of placeholder company logos + "30+ More" badge

### Why Vorten Studio
- 5 feature cards in a grid:
  1. Quality obsessed - "We operate with a single standard. If it's not world-class, it doesn't ship."
  2. Senior talent only - "You work directly with senior talent. No juniors. No outsourcing."
  3. 1:1 dev implementation - "Our senior frontend development team delivers true 1:1, pixel-perfect builds."
  4. Dedicated PM - "A single point of contact, fully invested in the project's success."
  5. Illustration, motion, and more - "Illustration and animation designed to make your product feel alive."

### Testimonials
- 3 testimonial cards with quotes, names, and company badges
- Use placeholder names/companies

### Tools We've Mastered
- Grid of tool/technology logos (use placeholder pills)

### Team
- Grid of team member cards with avatar placeholders, names, and roles
- "5+ More" expansion badge

### Pricing
- Two pricing columns:
  - Custom scope: $10,000+ (list features)
  - Design/dev retainer: $10,000/mo - Most Popular (list features)

### FAQs
- Accordion-style FAQ section with common questions

### Footer CTA
- "Let's build something cool" heading
- Same CTA buttons as hero
- Copyright: "© 2025 Vorten Studio"

## Design Specs
- Font: Inter (Google Fonts)
- Background: #f5f5f0 (warm light gray like the original)
- Dark text: #232323
- Accent: #FF4D00 (orange, like original)
- Cards: white with subtle rounded corners (1.5rem)
- Buttons: pill-shaped (fully rounded)
- Clean, modern, lots of whitespace
- Responsive (mobile-first)
- Smooth scroll behavior
- CSS animations for the showcase gallery marquee

## Important
- Single HTML file with everything inline
- Use Google Fonts for Inter
- Use placeholder gradients/colors for showcase images
- SVG icons inline where needed
- Make it pixel-perfect and professional
- The showcase gallery should auto-scroll horizontally like a marquee
